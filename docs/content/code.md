## Hello World
These are examples of **Hello World** application in multiple languages:


=== "C"
    ```c
    #include 
 
    int main(void){
        puts("Hello, world!");
    }
    ```

=== "Java"
    ```java
    public class HelloWorld {
        public static void main(String[] args) {
           System.print.out("Hello, World!");
        }
    }
    ```

=== "Python"
    ```python
    print("Hello, World!")
    ```
=== "Swift"
    ```swift
    println("Hello, world!")
    ```
=== "Go"
    ```go    	
    import "fmt"
    func main() {
        fmt.Println("hello world")
    }
    ```
